📖 대회 기록 관리 사용법

📋 전체 흐름

🔍 수집 가능 → ✏️ 검토 필요 → ✅ 완료

대회를 수집하면 검토 필요로 이동하고, 검토 후 저장하면 완료로 이동합니다.

🔍 1단계: 기록 수집

  • 수집 가능 탭에서 대회를 선택하면 자동으로 기록을 수집합니다.
  • 목록에 없는 대회는 하단 + 직접 추가에서 대회명·날짜를 입력하세요.
  • 이벤트 ID를 알고 있다면 이벤트 ID로 직접 검색을 사용하세요.

✏️ 2단계: 결과 검토

  • 검토 필요 탭에서 대회를 클릭하면 수집된 결과가 표시됩니다.
  • 이름이 노란색으로 표시된 행은 회원 매칭이 애매한 경우입니다. 직접 회원을 선택하거나 제외하세요.
  • DNF/기록 없음 행은 기본적으로 제외됩니다. 체크박스로 포함 여부를 조정할 수 있습니다.
  • PB 버튼을 눌러 개인 최고 기록 여부를 수동으로 조정할 수 있습니다.
  • 비고 열에 수상·특이사항 등을 메모할 수 있습니다.
  • 게스트(비회원) 기록은 게스트 행 추가로 직접 입력하세요.

💾 3단계: 기록 저장

  • 검토가 끝나면 기록 저장하기를 클릭합니다.
  • 최종 확인 화면에서 참가자 수·PB 목록을 확인하고 확정을 누릅니다.
  • 저장된 기록은 대회 기록 페이지에 즉시 반영됩니다.
  • 저장 후에도 수정하기를 눌러 재확정할 수 있습니다.
  • 완료 탭에서 대회를 클릭하면 저장된 기록을 확인하거나 재수정할 수 있습니다.
