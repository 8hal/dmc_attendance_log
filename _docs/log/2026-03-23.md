### 퍼널 (사람 기준)
```
검색 시작 17명 → 검색 완료 7명 (41%) → 저장 3명 (18%)
```

### 가설
| # | 가설 | 영향 구간 | 검증 방법 |
|---|------|----------|----------|
| 1 | 검색 실패/속도 | 시작→완료 59% 탈락 | search_error, search_no_result, search_cancel 로그 |
| 2 | 노출 꺼림 | 완료→저장 57% 탈락 | 피드백 수집, 공개 범위 설정 검토 |
| 3 | 동명이인 | 복합 | 페이스 표시 + 성별 필터 후 전환율 비교 |

### 핵심 인사이트
- **유입보다 안정화가 먼저.** 19명 들어왔는데 전환율 18%면, 새 사람 데려와봐야 같은 이유로 이탈.
- **데이터로 먼저, 설문은 나중에.** 개선 배포 후 1주일 로그 비교 → 수치 안 올라가면 그때 설문.
- **정합성 불일치는 대부분 표시 오차.** race_results가 SSOT이므로 실제 데이터 영향 없음.
