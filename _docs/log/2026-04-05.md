### 1. ohmyrace HTML 파싱 버그 수정

**문제 발견:**
- report.html에서 ohmyrace 118번 이벤트(군산 새만금 마라톤) 검색 시 결과 0건
- 브라우저에서는 정상 조회되나 스크래퍼는 실패

**근본 원인 분석:**
- ohmyrace 응답 HTML이 `<tr><td>` 테이블 구조가 아님
- 실제 구조: `<div class="name-card">` 기반
- 기존 파싱 로직이 테이블 구조를 전제로 작성됨

**해결:**
- `searchOhmyrace()` 함수 파싱 로직 전면 수정
- `.name-card` 기반으로 변경
- 이름, 배번, 거리, 기록, 순위(종합/성별), 성별 모두 추출

**테스트:**
```bash
# 로컬 테스트
node -e "김성한 검색" → 1건 정상 반환
  - 배번: 43410
  - 기록: 02:58:26
  - 종합: 14위, 성별: 4위
  - 성별: M
```
