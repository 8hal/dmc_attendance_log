## P0 트리아주 (실패 항목)

### 1. report.html > confirm 수정 기능 버그

**재현 스텝:**
1. report.html 완료 탭 열기
2. 특정 대회(예: "청라 하늘대교마라톤 2026") 선택 → "수정하기" 클릭
3. 기록 수정 또는 모두 삭제 시도

**문제:**
- 수정 시 1건 이상 필수: 모든 기록을 삭제하고 싶을 때 액션 없음
- 수정 저장 시 정상 동작하지 않음 (구체적 오류 미확인)

**우선순위:** P0 (운영 기능 직접 영향)

**권장 액션:**
1. 재현 스텝 상세화 (구체적 오류 메시지, 네트워크 요청 확인)
2. 코드 분석: report.html `confirm` 수정 로직, `update-record` API (있다면)
3. 최소 패치: "모두 삭제" 버튼 추가 또는 0건 저장 허용
