### 스크래퍼 피처 수집 구조 추가 (`79aa091`, v0.8.0)

이전까지는 소스별로 수집하는 필드가 들쭉날쭉했고, `ageGroupRank`/`splits`는 아무 소스도 수집하지 않았음.
각 소스 사이트를 직접 조사한 후 파싱 로직 구현.

| 소스 | gender | ageGroupRank | splits |
|------|:------:|:------------:|:------:|
| **spct** | 기존 ✅ | ✅ `.rank li` 3번째 항목 | ✅ Section 테이블 경과시간 |
| **smartchip** | ✅ Total_Rank URL `gender=남/여` | ❌ 없음 | ⬜ rawData 포맷 미확인 |
| **marazone** | ✅ `Sex` 필드 | ✅ `A_rank` ("660/1431") | ✅ `CP_01~CP_04_TIME` |
| **myresult** | 기존 ✅ | ❌ 없음 | ⬜ 추가 API 호출 필요 |

**버그 수정 포함**: marazone의 `rank_total`/`rank_gender` (없는 필드) → `O_rank`/`G_rank` 로 교체
