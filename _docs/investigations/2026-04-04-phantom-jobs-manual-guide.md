# Phantom Jobs 수동 처리 가이드

## 개요
report.html 완료 탭에서 18개 phantom jobs를 검토하고 처리합니다.

## 접속
```
https://dmc-attendance.web.app/report.html
```
→ "완료" 탭 클릭

## 처리 대상 (18개)

### 그룹 1: 즉시 다운그레이드 (12개)
아래 잡들은 **임시 검색 또는 테스트**이므로 실제 기록 없음:

```
테스트
무한도전 Run with 쿠팡플레이
제 7회 기장바다마라톤
2025 부안해변마라톤
2025 경기수원 국제하프마라톤
2025 JTBC 서울마라톤
2025 안양천단풍길마라톤
2025 슈퍼블루마라톤
제22회 여주 세종대왕 마라톤대회
2025 MBN 전국나주마라톤대회
2025철원DMZ국제평화마라톤
```

**처리 방법:**
1. 각 잡을 찾기 (완료 탭에서 검색)
2. "수정하기" 클릭
3. 확인: 기록 0건인지 확인
4. 0건이면: 그냥 닫기 (또는 삭제)

### 그룹 2: 수동 검증 필요 (7개)
정규 스크랩 잡이므로 실제 기록이 있을 수 있음:

```
1. 오사카마라톤
2. 부산비치울트라
3. 고구려마라톤
4. 2025 포항마라톤 챔피언십
5. 제35회 진주마라톤대회
6. 2025 부산바다마라톤
7. 제23회 희망드림동계국제마라톤
```

**처리 방법:**
1. report.html 완료 탭에서 대회 찾기
2. "수정하기" 클릭
3. 기록 확인:
   - **기록 있음**: 해당 회원 선택 → 재확정
   - **기록 없음** (0건): 
     - 옵션 A: 회원 재검색 시도 (다른 소스)
     - 옵션 B: 삭제 (기록이 정말 없는 경우)

## 스크립트 자동 처리 (선택)

그룹 1 (임시 검색 12개)만 자동으로 다운그레이드:

```bash

```
