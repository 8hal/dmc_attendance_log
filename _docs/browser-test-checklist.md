## Task 1: 기록 데이터 표시

### 테스트 시나리오

1. **확정된 기록 표시 확인**
   - [ ] 페이지 로드 후 참가자 목록 확인
   - [ ] "이미 확정" 문구가 **없는지** 확인
   - [ ] 기록 시간이 표시되는지 확인 (예: 3:45:22)
   - [ ] 순위가 표시되는지 확인 (예: 150위)
   - [ ] 코스 배지가 표시되는지 확인 (예: Half)

2. **DNS/DNF 배지 표시**
   - [ ] DNS 상태인 참가자에게 🚫 DNS 빨간 배지 표시 확인
   - [ ] DNF 상태인 참가자에게 ⚠️ DNF 노란 배지 표시 확인

3. **노이즈 제거 확인**
   - [ ] "배번 미입력" 문구가 **없는지** 확인
   - [ ] 배번이 있는 경우에만 "배번 XXX" 표시 확인

**Expected**: 기록 데이터가 명확하게 표시되고 노이즈가 제거됨

   - [ ] [확인] 클릭
   - [ ] 로딩 오버레이 표시: "DNF 처리 중..."
   - [ ] 성공 토스트 메시지: "✅ XXX 님 DNF 처리 완료"

3. **에러 처리**

## Task 4: 코스 변경

### 테스트 시나리오

1. **코스 변경 모달 열기**
   - [ ] ⋮ → [🔄 코스 변경] 클릭
   - [ ] 액션 모달 닫힘
   - [ ] 코스 변경 모달 열림
   - [ ] 현재 코스 표시: "현재: Half"

2. **코스 선택**
   - [ ] 드롭다운에 6개 옵션 표시 확인:
     - 풀 마라톤 (Full)
     - 하프 마라톤 (Half)
     - 10K
     - 5K
     - 30K
     - 3K
   - [ ] 현재 코스가 미리 선택되어 있는지 확인

3. **경고 메시지**
   - [ ] 노란색 경고 박스 표시: "⚠️ 코스 변경 시 기존 기록이 삭제되고 새 기록으로 저장됩니다."

4. **코스 변경 실행**
   - [ ] Full → Half 선택
   - [ ] [변경] 버튼 클릭
   - [ ] 확인 다이얼로그: "XXX 님 코스를 half로 변경하시겠습니까?"
   - [ ] [확인] 클릭
   - [ ] 로딩 오버레이: "코스 변경 중..."
   - [ ] 성공 토스트: "✅ XXX 님 코스 변경 완료"
   - [ ] 페이지 자동 새로고침
   - [ ] Half 배지 표시 확인

5. **같은 코스 선택 시**
   - [ ] 현재 코스와 같은 코스 선택
   - [ ] [변경] 클릭
   - [ ] 경고 토스트: "⚠️ 변경할 코스가 같습니다"

**Expected**: 코스 변경이 정상 작동하고 기존 기록 유지
