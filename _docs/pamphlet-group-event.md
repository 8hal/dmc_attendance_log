# 단체 대회 기록 관리 가이드

> **대상**: 동마클 운영진  
> **작성일**: 2026-04-13  
> **목적**: 단체로 신청한 대회의 기록을 자동으로 수집하고 확정하는 방법

---

## 🎯 단체 대회 기록 관리란?

동마클에서 **단체로 신청한 대회**의 경우, 참가자를 미리 등록해두면 **대회 후 자동으로 기록을 수집**하고 **한 번에 확정**할 수 있습니다.

**기존 방식**: 대회 후 한 명씩 검색 → 기록 확인 → 확정 (반복)  
**단체 대회 방식**: 참가자 미리 등록 → 대회 후 자동 수집 → 일괄 확정

## 📱 사용 화면

### 1. 단체 대회 관리 페이지
- **URL**: `https://dmc-attendance.web.app/group.html`

- **기능**: 단체 대회 등록, 참가자 선택, 기록 확정

- **접근**: 오너 비밀번호만 접근 가능
- **기능**: 기록 소스(사이트) 지정, 자동 스크랩 트리거

## 📋 단체 대회 등록 절차

### Step 1: 대회 등록
1. `group.html` 접속 → 운영자 비밀번호 입력
2. **"+ 단체 대회 등록"** 버튼 클릭
3. 대회명 검색 (예: "경기 마라톤")
4. 대회 선택 → 등록 완료

### Step 2: 참가자 선택
1. 등록된 대회 카드에서 **"참가자 선택"** 버튼 클릭
2. 회원 목록에서 참가자 체크 (또는 검색 후 Enter 키로 빠른 선택)
3. **"저장"** 클릭

> **Tip**: 회원이 목록에 없으면? **"+ 회원 관리"** 링크 → `admin.html`에서 회원 추가 → **"↺ 새로고침"** 버튼

### Step 3: 기록 소스 지정 (오너)
1. `ops.html` 접속 → 오너 비밀번호 입력
2. **"단체 대회 기록 소스 관리"** 섹션으로 스크롤
3. 대회 찾기 → 기록 사이트 선택 (예: smartchip)
4. 대회 번호 입력 (예: 202650000123)
5. **"저장 + 즉시 스크랩"** 버튼 클릭

> **기록 사이트 6종**: smartchip, myresult, spct, marazone, ohmyrace, manual

### Step 4: 기록 확정
**대회 다음 날 (자동 스크랩 후)**

1. `group.html` 접속
2. 대회 카드에서 **"기록 확인"** 섹션 펼치기
3. 결과 확인:
   - ✅ **자동 매칭**: 이름·기록 확인
   - ⚠️ **동명이인**: 라디오 버튼으로 선택
   - 🔴 **기록 없음**: DNS(기권) 또는 DNF(완주 실패) 선택

4. **"일괄 확정"** 버튼 클릭

---

## ⚙️ 자동 스크랩 일정

- **시간**: 매일 오후 3시 (KST)
- **대상**: 오늘 날짜 단체 대회 중 소스가 지정된 것
- **조건**: 
  - 참가자가 1명 이상 등록됨
  - 기록 소스(사이트 + 번호)가 입력됨
  - 아직 스크랩되지 않음

**예시**: 4월 20일 대회 → 4월 20일 오후 3시에 자동 스크랩

## 🔍 상태 표시

| 표시 | 의미 | 설명 |
|---|---|---|
| **대기** | 소스 미설정 | ops.html에서 기록 소스 입력 필요 |
| **스크랩 중** | 자동 수집 진행 중 | 1~2분 후 새로고침 |
| **✅ 완료** | 스크랩 완료 | 기록 확인 가능 |
| **⚠️ 부분 실패** | 일부만 수집됨 | 확인 후 재스크랩 또는 수동 입력 |
| **❌ 실패** | 스크랩 실패 | 소스 번호 확인 또는 재시도 |

## ❓ 자주 묻는 질문

### Q1. 참가자를 나중에 추가할 수 있나요?
**A**: 네, 언제든지 **"참가자 선택"** 버튼으로 추가 가능합니다. 단, 이미 스크랩이 완료된 경우 **ops.html에서 재스크랩** 필요합니다.

### Q2. 기록 소스 번호는 어디서 확인하나요?
**A**: 대회 공식 기록 사이트 URL에 표시됩니다.
- 예: `smartchip.co.kr/...?no=202650000123` → 번호는 `202650000123`

### Q3. 동명이인이 너무 많으면?
**A**: 나이, 기록, 동호회 정보 등을 참고해서 라디오 버튼으로 선택합니다. 불확실하면 해당 회원에게 직접 문의하세요.

### Q4. 자동 스크랩이 안 됐어요
**A**: 아래 체크리스트 확인:
- [ ] ops.html에서 소스(사이트 + 번호) 입력했는가?
- [ ] 참가자가 1명 이상 등록되어 있는가?
- [ ] 대회 날짜가 오늘인가? (자동 스크랩은 대회 당일 오후 3시)
- [ ] ops.html에서 **"저장 + 즉시 스크랩"** 버튼을 눌렀는가?

### Q5. 단체 대회를 삭제하려면?
**A**: group.html 대회 카드 우측 상단 **⋮ 더보기** → **삭제** 선택. 
- ⚠️ 대회만 삭제되고, 이미 확정된 기록은 유지됩니다.
