# 2026-04-15 단체 대회 상세 페이지 - 최종 확정사항

**작성일**: 2026-04-15  
**상태**: 기획 최종 승인 ✅

---

## 핵심 변경사항 (3가지)

### 1. ✅ 용어 변경
- **"갭 탐지" → "기록 매칭"**
- 이유: 운영자 친화적 용어, 직관적 이해
- 적용: UI 전체 (버튼, 레이블, 문서)

### 2. ✅ group.html 카드 간소화
**제거 (중복 기능):**
- 참가자 편집 버튼 → 상세 페이지에서만
- 기록 매칭 결과 전체 (동명이인 선택, 일괄 확정 포함) → 상세 페이지 전용

**추가 (요약 정보):**
- 매칭 현황 배지 (✅ 78명 ⚠️ 2명 🔴 5명)
- 확정 진행률 프로그레스 바 (0 / 85명)
- 카드 클릭 → group-detail.html 이동

**목적**: 명확한 역할 분리
- group.html: 목록 + 요약
- group-detail.html: 상세 관리 + 작업

### 3. ✅ 배번 입력 기능
- Phase 2 → **Phase 1 (필수)** 로 승격
- 위치: 각 참가자 row에 인라인 편집
- 목적: 동명이인 해소, 매칭 정확도 향상

## 페이지별 역할

### group.html (목록 + 요약)
```
┌─────────────────────────────────┐
│ 제24회 경기마라톤대회      ⋮   │
│ 📅 2026-04-19 · ✅ 스크랩 완료  │
├─────────────────────────────────┤
│ 참가자: [칩6개] +79명           │
├─────────────────────────────────┤
│ 처리 현황:                      │
│   ✅ 78명  ⚠️ 2명  🔴 5명      │
│   확정: 0 / 85명 (0%)           │
└─────────────────────────────────┘
  ↓ 클릭하면 상세로 이동
```

**기능:**
- 대회 카드 목록
- 매칭 현황 요약 (배지)
- 확정 진행률 (프로그레스 바)
- 대회 등록/삭제 (⋮ 메뉴)
- **카드 클릭 → 상세 페이지**

### group-detail.html (상세 관리 전용)
```
┌─────────────────────────────────┐
│ ← 단체 대회 관리                │
│ 제24회 경기마라톤대회           │
├─────────────────────────────────┤
│ 📊 통계 요약                    │
│ 등록 85명 / 확정 0명            │
│                                 │
│ ⚠️ 미처리 5명                   │
│ ████████░░ 80/85 (94%)          │
│ • 디모 (동명이인 선택 필요)     │
│ • SJ (DNS/DNF/직접입력 필요)    │
├─────────────────────────────────┤
│ 👥 참가자 목록 (85명)           │
│ [검색] [참가자 편집] [필터]     │
│                                 │
│ ✅ 라우펜더만 1:45:23           │
│    배번 12345  ✏️               │
│ ⚠️ 디모 (동명이인)              │
│    ◉ 3:28  ○ 4:12  [선택]      │
│ 🔴 SJ (기록 없음)               │
│    [DNS] [DNF] [직접 입력]      │
└─────────────────────────────────┘
┌─────────────────────────────────┐ Sticky
│ ⚠️ 미처리 5명  [일괄 저장] 🔒  │
└─────────────────────────────────┘
```

**기능:**
- 참가자 전체 목록 + 편집
- **배번 입력/수정** (인라인)
- 기록 매칭 결과 처리
- 동명이인 선택
- DNS/DNF/직접 입력
- 일괄 저장 (전원 선택 필수)
- 통계 (저장 완료 후)

  - Sticky footer

   - 대회 정보 + 매칭 결과 조회
   - 확정 진행 상태
   
2. **POST bulk-confirm**
   - 85건 일괄 저장 (idempotent)
   - Batch write (500건씩)
