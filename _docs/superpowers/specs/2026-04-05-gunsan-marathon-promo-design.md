# 군산 마라톤 "이 대회 뛰었나요?" 홍보 페이지 디자인

**작성일**: 2026-04-05  
**버전**: 1.0

---

## 1. 목표 및 배경

### 목표
동마클 회원 중 군산 새만금 마라톤 참가자들에게 "이 대회 뛰었나요?" 신규 기능을 알리고, **즉시 사용을 유도**한다.

### 타겟
- **동마클 회원만** (내부 배포)
- 군산 새만금 마라톤 2026 참가자

### 배포 채널 (복합)
1. **네이버 카페 게시물** — 게시물에 페이지 링크 공유
2. **카카오톡 직접 배포** — 단체방 또는 개별 메시지로 URL 전달
3. **기존 시스템 통합** — my.html에 배너 형태로 노출

### 성공 기준
**클릭률 (CTR)** 극대화:
- "내 기록 확인하기" 버튼 클릭
- "이 대회 뛰었나요?" 페이지에서 "내 기록" 또는 "아니오" 버튼 클릭

2. ✅ **동마클 브랜딩 유지** — 기존 팜플렛 스타일과 일관성

### 4.1 헤드라인 (Hero Section)

**메인 제목**:
```
🏃‍♂️ 군산 마라톤, 뛰셨나요?
```

**부제목**:
```
새로운 기능으로 1분 안에 내 기록을 자동으로 찾아드립니다
```

**특징**:
- 직접적, 질문형
- 이모지로 시선 집중
- 부제목에서 가치 제안 명확히 ("자동으로", "1분")

---

### 4.2 기능 시연 영역 (Device Mockup)

**핵심 요소**: 실제 기능 화면 스크린샷

**구성**:
1. **폰 프레임 mockup** (선택: iPhone 또는 안드로이드 스타일)
2. **내부 스크린샷**: 실제 "이 대회 뛰었나요?" 기능 화면
   - 대회 정보 (예: "제 33회 3.15 마라톤")
   - 기록 표시 (예: "10K 00:50:50")
   - **"내 기록"** 버튼 (초록색)
   - **"아니오"** 버튼

**스타일**:
- 스크린샷을 폰 프레임 안에 배치
- 배경: 그라데이션 또는 단색 (초록 계열 추천)
- 중앙 정렬
- 화면 너비의 80% 정도 크기

**이유**:
- 사용자가 "어떻게 생겼는지" 즉시 이해
- 신뢰도 증가 (실제 화면 보여줌)
- 모바일 앱 랜딩 페이지 표준 패턴

---

### 4.3 기능 설명 (Value Proposition)

**간단한 설명** (3줄 이내):
```
대회 후 내 기록을 찾기 위해 여러 사이트를 뒤지셨나요?
이제 동마클이 자동으로 찾아드립니다.
이름만 입력하면 끝!
```

**스타일**:
- 중앙 정렬
- 폰트 크기: 0.95em ~ 1em
- 색상: #666 (본문)
- Line-height: 1.8 (가독성)

---

### 4.4 CTA 버튼

**문구**:
```
지금 확인하기 →
```

**스타일**:
- **색상**: 초록색 계열 (`#27ae60` 또는 기존 "내 기록" 버튼과 동일)
  - 이유: 기존 시스템의 "내 기록" 기능과 색상 일관성 유지
- **크기**: 화면 너비의 85%
- **패딩**: 상하 1.2rem
- **폰트**: 1.2em, 700 weight
- **보더**: 없음, border-radius 12px
- **화살표**: → (행동 유도)
- **그림자**: `box-shadow: 0 4px 12px rgba(39, 174, 96, 0.3)` (입체감)

**부가 문구** (버튼 아래):
```
1분이면 끝 · 회원가입 불필요
```

---

### 4.5 추가 요소 (Optional)

**간단한 사용법 (3단계)**:
```
1️⃣ 이름 입력
2️⃣ 대회 선택
3️⃣ 결과 확인
```

- 아이콘 + 짧은 텍스트
- 폰 mockup 아래 또는 CTA 버튼 아래 배치
- 선택적 (페이지가 너무 길어지면 제거)

**동마클 로고** (하단):
- 작게 배치 (브랜딩)
- 페이지 최하단 중앙

## 6. 기술 명세

### 파일명
`gunsan-marathon-promo.html`

### 기술 스택
- **HTML5** (단일 파일)
- **Inline CSS** (외부 의존성 없음)
- **반응형 디자인** (viewport meta 태그)

### 필수 요소
1. `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
2. 모바일 우선 CSS (max-width 기준)
3. 터치 영역 최소 44x44px (버튼)

### 주요 자산

**기능 스크린샷**:
- 파일명: `gunsan-feature-screenshot.png` (또는 실제 화면 캡처)
- 크기: 최소 750x1334px (iPhone 8 Plus 해상도)
- 내용: "이 대회 뛰었나요?" 화면 (군산 마라톤 데이터 표시)
- 위치: `assets/` 또는 Firebase Storage

**폰 프레임 mockup** (옵션):
- CSS로 구현 가능 (border-radius + box-shadow)
- 또는 PNG 오버레이 사용

**동마클 로고**:
- 파일명: 기존 팜플렛에서 재사용
- 크기: 작게 (하단 배치용)

### CTA 버튼 링크
**타겟 URL**: `my.html` (또는 "이 대회 뛰었나요?" 직접 검색 페이지)

**파라미터** (옵션):
- `?source=gunsan-promo` — 유입 경로 추적용
- `?event=gunsan-2026` — 대회 자동 선택 (가능하다면)
