① 카페 공지 붙여넣기

네이버 카페 정모 공지 본문 텍스트, 또는 글 API JSON(contentHtml 포함)을 붙여넣으면 됩니다. 이미지·URL만으로는 파싱되지 않습니다.

카페 글 화면 콘솔에서 필요한 부분만 복사

Chrome DevTools 콘솔에 붙여넣고 Enter → 클립보드에 최소 JSON이 복사됩니다. 그다음 아래 칸에 붙여넣기하세요.

A. 글 본문 화면 (가장 간단)

copy((()=>{const el=document.querySelector(".se-main-container")||document.querySelector(".se-viewer");if(!el){alert("본문을 찾지 못했습니다. 정모 공지 글 화면에서 실행하세요.");return;}const subject=(document.querySelector(".title_text,.ArticleTitle,.article_tit,.tit_area h3")||{}).textContent?.trim()||document.title;return JSON.stringify({result:{article:{subject,contentHtml:el.outerHTML}}});})())

B. Network 응답을 temp1로 저장한 경우 (응답 우클릭 → Store as global variable)

copy(JSON.stringify({result:{article:{subject:temp1.result.article.subject,contentHtml:temp1.result.article.contentHtml}}}))
