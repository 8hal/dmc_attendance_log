🎯 단체 대회 기록 관리란?

동마클에서 단체로 신청한 대회의 경우, 참가자를 미리 등록해두면 대회 후 자동으로 기록을 수집하고 한 번에 확정할 수 있습니다.

기존 방식
대회 후에만 검색 → 기록 확인 → 확정
✨ 단체 대회 방식
참가자 미리 등록 → 대회 후 자동 수집 → 일괄 확정
📌 참고: 기록 소스(사이트) 지정은 시스템 담당자가 별도로 진행합니다.

📋 단체 대회 등록 절차

대회 등록

  1. group.html 접속 → 운영자 비밀번호 dmc2008 입력
  2. "+ 단체 대회 등록" 버튼 클릭
  3. 대회명 검색 (예: "경기 마라톤")
  4. 대회 선택 → 등록 완료

참가자 선택

  1. 등록된 대회 카드에서 "참가자 선택" 버튼 클릭
  2. 회원 목록에서 참가자 체크 (또는 검색 후 Enter 키로 빠른 선택)
  3. "저장" 클릭
💡 Tip: 회원이 목록에 없으면?
"+ 회원 관리" 링크 → admin.html에서 회원 추가 → "↺ 새로고침" 버튼

기록 확정 (대회 다음 날)

  1. group.html 접속
  2. 대회 카드에서 "기록 확인" 섹션 펼치기
  3. 결과 확인:
    • ✅ 자동 매칭: 이름·기록 확인
    • ⚠️ 동명이인: 라디오 버튼으로 선택
    • 🔴 기록 없음: DNS(기권) 또는 DNF(완주 실패) 선택
  4. "일괄 확정" 버튼 클릭

⚙️ 자동 스크랩 일정

실행 시간
매일 오후 3시 (KST)
실행 대상
오늘 날짜 단체 대회 중 소스가 지정된 것
실행 조건
✅ 참가자가 1명 이상 등록됨
✅ 기록 소스(사이트 + 번호)가 입력됨
✅ 아직 스크랩되지 않음

예시: 4월 20일 대회 → 4월 20일 오후 3시에 자동 스크랩

❓ 자주 묻는 질문

Q1. 참가자를 나중에 추가할 수 있나요?

A: 네, 언제든지 "참가자 선택" 버튼으로 추가 가능합니다.

Q2. 동명이인이 너무 많으면?

A: 나이, 기록, 동호회 정보 등을 참고해서 라디오 버튼으로 선택합니다. 불확실하면 해당 회원에게 직접 문의하세요.

Q3. 자동 스크랩이 안 됐어요

A: 시스템 담당자에게 문의하세요. 기록 소스 설정이 필요할 수 있습니다.

Q4. 단체 대회를 삭제하려면?

A: group.html 대회 카드 우측 상단 ⋮ 더보기 → 삭제 선택.
⚠️ 대회만 삭제되고, 이미 확정된 기록은 유지됩니다.
